Asistente Inteligente de Evaluación de Dificultades de Aprendizaje

Este proyecto desarrolla un modelo de Machine Learning capaz de predecir el nivel de riesgo académico de un estudiante a partir de su comportamiento digital en plataformas educativas. Analiza variables como tiempos de respuesta, tasa de errores, consistencia e intentos fallidos para clasificar a cada estudiante en:
Bajo, Medio o Alto riesgo.

Objetivo del Proyecto

Implementar un sistema que permita la detección temprana de posibles dificultades de aprendizaje mediante técnicas de inteligencia artificial, facilitando la intervención docente oportuna.

Herramientas Utilizadas

Python 3

Pandas

NumPy

scikit-learn

Random Forest Classifier

Google Colab

Descripción Técnica

El dataset fue generado artificialmente con 300 registros que representan diferentes comportamientos estudiantiles.
Posteriormente se entrenó un modelo Random Forest con un accuracy del 85%.

Se incluye una demostración que permite predecir el riesgo de un nuevo estudiante en tiempo real.

Resultados

Accuracy del modelo: 85%

Clasificación: Bajo / Medio / Alto

Funcionamiento en tiempo real: Sí

Interpretabilidad: Importancia de características disponible

Cómo ejecutar el proyecto

Descargue el notebook notebook.ipynb.

Ábralo en Google Colab.

Ejecute las celdas en orden.

Ingrese los valores de un nuevo estudiante para obtener su nivel de riesgo.
